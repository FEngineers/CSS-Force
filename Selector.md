# [X] element selector
```css
p {color: black;}
h1 {font-weight: bold;}
```

# class selector
```css
.date-posted {color: #ccc;}
```

# id selector
```csss
#intro {font-weight: bold;}
```

# 通配符选择器
```css
* {
 margin: 0;
 padding: 0;
}
#container * {
 border: 1px solid black;
}
```
I'd advise you to never use this in production code. It adds too much weight on the browser, and is unnecessary.
